# pip版本太低

![image-20200620230045697](../images/20200620pip/1.png)

使用pip安装某个包时，容易报这个pip版本的错。

解决方案：升级命令：python -m pip install --upgrade pip

![image-20200620230151764](../images/20200620pip/2.png)

已经成功升级，再次安装，成功！
